# 坦克大战

经典坦克大战 HTML5 Canvas 游戏

## 游戏特性

- 玩家坦克由键盘控制（WASD 移动，空格射击）
- 敌方坦克具有 AI 行为
- 砖墙和钢墙障碍物
- 基地保卫玩法
- 多关卡设计
- 计分系统

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 | 射击 |
| P | 暂停/继续 |
